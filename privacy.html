Политика конфиденциальности

Приложение «Агент по сбору чемодана» — это личный список вещей для поездки. Ниже честно и просто о том, какие данные обрабатываются.

Какие данные собираются

Приложение не требует регистрации, не запрашивает имя, телефон, e-mail или геолокацию. Всё, что вводит пользователь (город назначения, тип поездки, отмеченные пункты списка, «маст-хэвы»), сохраняется только локально на устройстве пользователя (localStorage браузера) и никуда не передаётся и не хранится на серверах разработчика.

Аналитика

Для понимания того, какие функции используются, подключена Яндекс.Метрика — она фиксирует обезличенные технические события (какая функция была нажата, тип устройства, страна) без сбора персональных данных, вводимых в форму. Подробнее о том, как обрабатывает данные сама Яндекс.Метрика: yandex.ru/legal/confidential.

Ссылки на сторонние сервисы

В списке вещей есть ссылки на партнёрские сервисы (Купер, Самокат, СберСтрахование, Okko, Звук) — переход по ним открывает сайт соответствующего сервиса, у которого своя политика конфиденциальности. Разработчик приложения не получает и не видит никаких данных о действиях пользователя на этих сайтах.

Удаление данных

Поскольку все данные хранятся локально, для их удаления достаточно очистить данные сайта/приложения в настройках браузера или устройства.
